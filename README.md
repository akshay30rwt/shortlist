# Shortlist

A full-stack applicant tracking system for small hiring teams — manage job openings, track candidates through a recruitment pipeline, and let candidates apply through a public job board.

**Status:** Early development.

## Tech Stack
Next.js (App Router) · TypeScript · Tailwind CSS · shadcn/ui · MongoDB · Mongoose · Auth.js · Zod

## Local Development
```
npm install
npm run dev
```
